Frontend (React) for Todo app

1) Install deps and start dev server:
   cd frontend
   npm install
   npm start

2) The frontend proxies API requests to the backend at http://localhost:8080 via the "proxy" field in package.json.
3) Dev UI runs at http://localhost:3000

Notes: This is a minimal Create React App-compatible project. If you prefer Vite or another bundler, swap files accordingly.
